Persönliches, kuratiertes Threat-Intelligence-Repo für Android-Malware.

**Ziel:** Früherkennung von RATs, Bankern, Minern und Spyware auf meinem Samsung A55.

### Inhalt
- `threats.txt` – einfache Prozess-/Paketnamen
- `android_malware.json` – strukturierte IOCs (SHA256, Family, Description)
- `yara_rules/` – YARA-Signaturen

### Auto-Update
GitHub Action holt täglich neue IOCs von vertrauenswürdigen Quellen.

### Nutzung
Wird von meinem IrsanAI 360 Smartphone SHIELD (Termux) automatisch gezogen.
